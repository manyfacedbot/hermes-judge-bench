# Problem 001 — Sum of Multiples

Find the sum of all natural numbers below 1000 that are multiples of 3 or 5.

Your script must print a single integer to stdout when run with `python solution.py`.

**Scoring:** `correctness / (1 + total_tokens / 10000)`, where `correctness` is 1.0 if the printed integer matches the expected answer exactly and 0.0 otherwise. `total_tokens` is the sum of agent and judge tokens across the /goal loop — a wrong-or-late answer scores 0; an extra 10,000 tokens halves your score.
