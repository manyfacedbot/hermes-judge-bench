# Problem 054 — Poker Hands

The file `poker.txt` (in the same directory as `solution.py`) contains 1,000 hands dealt to two players. Each line contains ten cards (separated by spaces): the first five are Player 1's cards and the last five are Player 2's cards.

Card notation:
- **Value:** 2, 3, 4, 5, 6, 7, 8, 9, T, J, Q, K, A (low to high)
- **Suit:** C (Clubs), D (Diamonds), H (Hearts), S (Spades)

Hand rankings (highest to lowest):
1. Straight Flush
2. Four of a Kind
3. Full House
4. Flush
5. Straight
6. Three of a Kind
7. Two Pairs
8. One Pair
9. High Card

Rules:
- If two players have the same hand rank, compare by highest card in the ranking group, then by kicker cards
- Every hand has a clear winner (no full ties)
- A-2-3-4-5 counts as a straight (wheel), with Ace playing low

Your script must:
1. Read `poker.txt` from the same directory as `solution.py`
2. Count how many hands **Player 1** wins
3. Print that count as a single integer to stdout

Run with: `python solution.py`

**Scoring:** Correct answer = 1. Wrong answer = 0. Score = correctness / log2(tokens_used).
