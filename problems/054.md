# Problem 054 — Poker Hands

The file `poker.txt` is in the problems data directory (the runner tells you the exact path in its system prompt — typically the same directory as the problem .md). It contains 1,000 hands dealt to two players. Each line contains ten cards (separated by spaces): the first five are Player 1's cards and the last five are Player 2's cards.

Card notation:
- **Value:** 2, 3, 4, 5, 6, 7, 8, 9, T, J, Q, K, A (low to high)
- **Suit:** C (Clubs), D (Diamonds), H (Hearts), S (Spades)

Hand rankings (highest to lowest):
1. Straight Flush
2. Four of a Kind
3. Full House
4. Flush
5. Straight
6. Three of a Kind
7. Two Pairs
8. One Pair
9. High Card

Rules:
- If two players have the same hand rank, compare by highest card in the ranking group, then by kicker cards
- Every hand has a clear winner (no full ties)
- A-2-3-4-5 counts as a straight (wheel), with Ace playing low

Your script must:
1. Read `poker.txt` from the problems data directory (path given to you by the runner)
2. Count how many hands **Player 1** wins
3. Print that count as a single integer to stdout

Run with: `python solution.py`

**Scoring:** `correctness / (1 + total_tokens / 10000)`, where `correctness` is 1.0 if the printed integer matches the expected answer exactly and 0.0 otherwise. `total_tokens` is the sum of agent and judge tokens across the /goal loop.
