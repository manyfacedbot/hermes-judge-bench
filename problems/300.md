# Problem 300 — Archipelago

You are playing a strategy game called **Archipelago** against another agent.

## The Game

**Board:** 10×10 grid, initially empty.

**Players:** Red and Blue. Each turn, a player places one stone on any empty cell.

**Islands:** A group of orthogonally connected stones of the same colour is an **island**.

**Scoring:** At the end of the game, each player scores points equal to the **total size of all their islands** (i.e. total number of their stones on the board). Additionally, any empty cell that is completely surrounded (orthogonally) by a single player's stones — reachable only through that player's stones using flood-fill from outside the board — scores **2 bonus points** for that player.

**End condition:** The game ends after each player has placed **30 stones** (60 total moves). Highest score wins. Draws are possible.

---

## Your Task

Write a Python bot that plays Archipelago. Your bot must implement:

```python
def choose_move(board: list[list[str]], my_colour: str, move_number: int) -> tuple[int, int]:
    """
    board:       10×10 grid. board[row][col] is 'R', 'B', or '' (empty).
    my_colour:   'R' or 'B'
    move_number: which move this is for you (1-indexed, 1..30)
    Returns:     (row, col) — must be a valid empty cell
    """
```

Write your bot to `/tmp/solution.py`. It must define `choose_move` as above.

---

## Tournament Structure

This is a **20-round tournament** against another agent (using a different judge configuration).

- Each round is **best of 3 games**. Round winner = agent that wins 2 or more of the 3 games.
- After each round, you will see the **full game results**: move histories, scores, and which games you won or lost.
- You have a **total token budget of 40,000 tokens** across all 20 rounds — once spent, it is gone.
- The agent that spends **fewer tokens in a given round** gets to move **first** (Red) in that round's games. If equal or both spend 0, the agent that went second last round goes first this round.
- You do NOT see your opponent's source code — only game results.

**Tournament winner** = agent that wins the most rounds out of 20.

---

## Scoring for this benchmark

```
quality  = rounds_won / 20
tokens   = total tokens spent across all rounds
score    = quality / (1 + tokens / 10000)
```

**Strategy:** A bot that wins 15/20 rounds spending 8,000 tokens beats one that wins 16/20 spending 18,000 tokens. Spend budget where it counts — a well-designed bot early, targeted patches after losses.

## Rules

- No web search.
- Your bot must run in under 5 seconds per move.
- No external libraries beyond Python stdlib.
- `/tmp/solution.py` is overwritten each round — write a complete, self-contained file every time.
- If your bot crashes or times out on a move, that move is chosen randomly for you.
