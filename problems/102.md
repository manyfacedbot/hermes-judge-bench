# Problem 102 — Poetry Compression: Ode and Elegy

You are competing against other agents to design the best losslessly invertible encoding for 19th century English poetry.

**Category:** Odes and elegies — longer meditative poems in elevated diction, including Keats's great odes, Shelley's elegies, Arnold's memorial verse, and Tennyson's In Memoriam sections.

**Task:**
Design an encoding system — a language, scheme, or algorithm — that can:
1. Encode any poem from this category into a compressed representation
2. Decode that representation back to the exact original text (lossless round-trip)
3. Achieve the highest possible compression ratio on unseen poems from this category

**What to build:**
Write a Python script to /tmp/solution.py that:
- Defines an `encode(text: str) -> str` function
- Defines a `decode(encoded: str) -> str` function
- Verifies round-trip correctness: `decode(encode(poem)) == poem` for all test poems
- Prints a JSON object to stdout: `{"compression_ratio": <float>, "verified": true}`

The test poems are available at: /home/hermes/hermes-judge-bench/problems/corpus/ode_and_elegy.json
Compression ratio = mean(len(original) / len(encoded)) across all poems in the corpus.

**Scoring:** `compression_ratio / (1 + tokens_used / 10000)`
where tokens_used = total inference tokens spent (input + output) across your entire session.
More compression = better. More tokens spent = penalized linearly. There is no hard token limit — spend as many as you need, but each 10,000 tokens halves your effective score.

**You are competing against other agents on this same problem.** Your score is absolute but ranking is relative. A quick correct solution that scores 1.5 beats a slow elaborate solution that scores 1.4.

**Rules:**
- No web search
- Your encode/decode must run on the current machine with no external dependencies beyond Python stdlib
- The eval will test your encoder/decoder against a random sample of complete poems from this category — never fragments
- Write your script to /tmp/solution.py
- When confident, stop and say DONE
